Conference Framework
=====================

這個專案是為了讓大家能夠更快速且成功的辦一個年會，希望製造出一個屬於Conference的Framework。

## 內容
這個專案將舉辦活動分成活動前、活動中與活動後3個階段的經營，每個階段依照組別分別列出該階段需要達成的目標。

### 活動前
- 議程
- 場地
- 資訊系統(APP)
    - CCIP
- 行政
- 財務
- 食物
- 印刷
- 設計
- 人事
- 行銷/贊助

### 活動後

## 大家來填坑
我們希望這個專案可以推廣到各Conference，讓大家可以把辦活動的經驗傳承，幫助更多有熱忱辦活動，卻缺乏經驗的人可以順利的舉辦活動。
